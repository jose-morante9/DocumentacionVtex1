Guía completa sobre Edition App en VTEX

Pasos para crear tu repositorio

1 Instalar VTEX Toolbelt

Primero necesitas tener instalado el CLI de VTEX:

npm install -g vtex
2 Iniciar sesión en tu cuenta

Autentícate con tu cuenta de VTEX:

vtex login {nombre-de-tu-cuenta}
3 Crear directorio del proyecto

Crea una carpeta para tu nueva app:

mkdir my-edition-app
cd my-edition-app
4 Inicializar proyecto Edition App

Usa el comando init y selecciona "edition app":

vtex init

Cuando aparezca el menú, selecciona "edition app"

5 Configurar manifest.json

Edita el archivo manifest.json:

{
  "name": "blackfriday-edition",
  "vendor": "mitienda",
  "version": "0.0.1",
  "title": "Black Friday Edition",
  "builders": {
    "edition": "0.x"
  }
}
6 Crear archivo edition/apps.json

Define qué apps se instalarán:

{
  "apps": [
    "vtex.store-theme@5.x",
    "vtex.store@2.x"
  ]
}
7 Link para desarrollo
vtex link
8 Publicar la app
vtex publish
Tip: Siempre prueba en workspace antes de producción.

Edition App gestiona versiones de tu tienda

En términos simples: Es como tener múltiples configuraciones predefinidas que activas según necesites.

Ejemplo con Theme Store

Tienda de ropa:

  • Normal: Theme corporativo estándar
  • Black Friday: Activas edition que cambia:
    • Theme oscuro (negro/amarillo)
    • Banner de ofertas
    • Contador de tiempo
    • Apps de descuentos
  • Navidad: Edition navideña:
    • Theme rojo y verde
    • Animación de nieve
    • Productos navideños
Ventajas:
  • Rapidez: Todo cambia con un comando
  • Seguridad: Pruebas en workspace
  • Reversible: Vuelves fácilmente
  • Organizado: Todo en una app
Cómo funciona:
vtex use blackfriday --edition mitienda.blackfriday@1.0.0

VTEX automáticamente:

  1. Instala apps de edition/apps.json
  2. Aplica configuraciones
  3. Actualiza theme store
  4. ¡Tu tienda está lista!

Comandos principales

Activar Edition
vtex use {workspace} --edition {vendor}.{app}@{version}

Ejemplo: vtex use bf --edition mitienda.bf-edition@1.0.0

Ver editions
vtex edition get
Cambiar edition
vtex edition set {vendor}.{app}@{version}
Promover a master
vtex workspace promote
Importante: Siempre prueba en workspace antes de producción.

Instalación de React en VTEX

1 Crear app con React

Inicializa una nueva app con soporte React:

vtex init
? Choose where to start: react
2 Configurar manifest.json

Agrega React en los builders:

{
  "name": "my-react-app",
  "vendor": "mystore",
  "version": "0.0.1",
  "builders": {
    "react": "3.x",
    "store": "0.x"
  },
  "dependencies": {
    "vtex.store": "2.x",
    "vtex.store-header": "2.x"
  }
}
3 Instalar dependencias React

Navega a la carpeta react/ e instala:

cd react
yarn install

También puedes usar npm install

4 Estructura de carpetas

Tu proyecto debe verse así:

my-react-app/
├── manifest.json
├── react/
│ ├── package.json
│ ├── tsconfig.json
│ └── components/
│ └── MyComponent.tsx
└── store/
└── interfaces.json
5 Configurar interfaces.json

Define tu componente en store/interfaces.json:

{
  "my-component": {
    "component": "MyComponent"
  }
}
6 Link y desarrollo
vtex link

Ahora puedes desarrollar con hot reload de React

Tip Pro

VTEX usa React + TypeScript por defecto. Aprovecha el tipado para evitar errores.

Usa vtex.render-runtime para componentes que necesiten interactuar con el store framework.

Nota: React en VTEX es diferente a React tradicional. Usa los hooks y componentes de VTEX IO.

Tu primer componente React

1 Crear componente HelloWorld.tsx

En react/components/HelloWorld.tsx:

import React from 'react'

interface HelloWorldProps {
  name?: string
}

const HelloWorld: React.FC<HelloWorldProps> = ({ name = 'VTEX' }) => {
  return (
    <div style={{
      padding: '20px',
      backgroundColor: '#F71963',
      color: 'white',
      borderRadius: '8px',
      textAlign: 'center'
    }}>
      <h1>¡Hola {name}!</h1>
      <p>Mi primer componente en VTEX IO</p>
    </div>
  )
}

export default HelloWorld
2 Configurar interfaces.json

En store/interfaces.json:

{
  "hello-world": {
    "component": "HelloWorld",
    "props": {
      "name": {
        "type": "string",
        "default": "VTEX"
      }
    }
  }
}
3 Usar en tu theme

Agrega a tu theme en cualquier bloque JSON:

{
  "store.home": {
    "blocks": [
      "hello-world"
    ]
  },
  "hello-world": {
    "props": {
      "name": "Mundo"
    }
  }
}
4 Link y visualizar
vtex link

Abre tu tienda en el workspace:

https://{workspace}--{account}.myvtex.com

Agregar a tu Edition App

5 Publicar tu app

Primero publica la app con el componente:

vtex publish
vtex install mystore.my-react-app@0.0.1
6 Agregar a edition/apps.json

En tu Edition App, incluye tu app custom:

{
"apps": [
"vtex.store-theme@5.x",
"vtex.store@2.x",
"mystore.my-react-app@0.0.1"
]
}
7 Usar en tu Theme Store

En tu theme (store/blocks/home.jsonc):

{
"store.home": {
"blocks": [
"hello-world#custom"
]
},
"hello-world#custom": {
"props": {
"name": "Mi Tienda VTEX"
}
}
}
8 Activar la Edition
vtex use production --edition mystore.my-edition@1.0.0

Tu componente ahora está disponible en toda la tienda

Flujo completo
  1. Crear app React con tu componente
  2. Publicar app (vtex publish)
  3. Agregar a Edition (edition/apps.json)
  4. Publicar Edition (vtex publish)
  5. Usar en Theme (blocks JSON)
  6. Activar Edition (vtex use)
